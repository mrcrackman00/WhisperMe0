Accessibility Commitment

WhisperMe is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. These guidelines explain how to make web content more accessible to people with disabilities, and more user-friendly for everyone.

Screen Reader Support

WhisperMe is designed to work with assistive technologies including screen readers. We follow best practices for accessibility including:

  • Semantic HTML elements for clear document structure
  • ARIA labels and roles for interactive elements
  • Alt text on all meaningful images and icons
  • Descriptive link text that makes sense out of context
  • Audio content paired with transcripts where possible

We test with popular screen readers including NVDA, JAWS, and VoiceOver.

Keyboard Navigation

All functionality on WhisperMe can be accessed using a keyboard alone. This includes:

  • All navigation menus and links are keyboard accessible
  • Visible focus indicators on all interactive elements
  • Logical tab order that follows the visual layout
  • Skip navigation links to bypass repetitive content
  • No keyboard traps — users can always navigate away from any component

Accessibility Features

WhisperMe includes the following accessibility features:

  • Voice-first design: Our platform is built around audio, making it naturally accessible to users who prefer voice interaction
  • Colour contrast: We maintain a minimum contrast ratio of 4.5:1 for all text content
  • Resizable text: All text can be resized up to 200% without loss of functionality
  • Reduced motion: We respect the prefers-reduced-motion media query to limit animations for users who request it
  • Dark/light mode: Platform adapts to system colour scheme preferences

Contact for Accessibility Issues

We welcome your feedback on the accessibility of WhisperMe. If you experience accessibility barriers or have suggestions for improvement, please contact us:

  • Email: accessibility@whisperme.co
  • General support: hello@whisperme.co

We aim to respond to accessibility feedback within 2 business days. We take all reports seriously and are committed to making WhisperMe better for everyone.
